#### 基础用法

<ClientOnly>
  <sticky-demos></sticky-demos>
</ClientOnly>


#### 示例代码
```html
    <!-- ... -->
    <p>段落10</p>
    <b-sticky top="100">
      <b-button>粘滞按钮</b-button>
    </b-sticky>
    <p>段落11</p>
    <!-- ... -->
```

#### API
#### Message props
属性| 说明 | 类型 | 默认值
---|---|---|---
top | 距离页面顶部粘滞的距离	 | String / Number | 0
